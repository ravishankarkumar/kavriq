# Communities

These AI/ML communities support the AI/ML in Rust tutorial, fostering discussion and collaboration for **Core ML**, **Deep Learning**, **Advanced Topics**, and **Projects** like **Customer Churn Prediction** and **Image Classification**. They align with Rust's ML ecosystem (`linfa`, `tch-rs`, `polars`) for efficient AI/ML development.

- **r/MachineLearning**: Reddit AI/ML forum. [reddit.com/r/MachineLearning](https://www.reddit.com/r/MachineLearning)
- **AI Alignment Forum**: AI safety discussions. [alignmentforum.org](https://www.alignmentforum.org)
- **Kaggle Community**: Data science competitions. [kaggle.com](https://www.kaggle.com)
- **Towards Data Science**: Medium AI/ML blog community. [towardsdatascience.com](https://towardsdatascience.com)
- **Fast.ai Forums**: Deep learning community. [forums.fast.ai](https://forums.fast.ai)
- **PyTorch Discuss**: Deep learning framework forum. [discuss.pytorch.org](https://discuss.pytorch.org)
- **Hugging Face Community**: NLP and transformers hub. [huggingface.co/community](https://huggingface.co/community)
- **DeepLearning.AI Discord**: AI/ML learning server. [discord.gg/deeplearningai](https://discord.gg/deeplearningai)
- **ML Collective**: AI/ML research community. [mlcollective.org](https://mlcollective.org)
- **Rust ML Working Group**: Rust AI/ML development. [github.com/rust-ml](https://github.com/rust-ml)
- **Reddit r/learnmachinelearning**: Beginner ML forum. [reddit.com/r/learnmachinelearning](https://www.reddit.com/r/learnmachinelearning)
- **NeurIPS Community**: AI/ML conference discussions. [neurips.cc](https://neurips.cc)
- **AI Village**: AI security community. [aivillage.org](https://aivillage.org)
- **Women in Machine Learning (WiML)**: Diversity in AI/ML. [wimlworkshop.org](https://wimlworkshop.org)
- **Papers with Code Community**: ML research and code. [paperswithcode.com/community](https://paperswithcode.com/community)
